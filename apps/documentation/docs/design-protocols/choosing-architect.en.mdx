---
title: Choosing Architect or Architect Classic
navOrder: 9
---

Architect is the protocol designer for Network Canvas — the tool you use to build the interview protocols that run in Interviewer and Fresco. It comes in two versions: a web app which runs in your browser and can be installed for offline use, and **Architect Classic**, the older downloadable desktop application (Windows, macOS, Linux). Throughout this documentation, "Architect" on its own means the web app, and "Architect Classic" always refers to the download. They share the same interface and protocol-building workflow, but they are **not interchangeable** — each builds a different version of the protocol file format ("schema"), and that version determines which app you can run your interviews in. The table below compares them; which you should use depends mainly on where you will run your interviews.

<KeyConcept title="Not sure which to use?">

Choose **Architect** for new studies — it's the default and needs no setup. Its schema 8 protocols run in **Fresco** and in the new **Interviewer** app.

Choose **Architect Classic** only if you need to keep using **Interviewer Classic**, the older desktop and tablet app. It builds schema 7 protocols and cannot open protocols built in Architect.

[Open Architect →](https://architect.networkcanvas.com)

</KeyConcept>

|                   | Architect                                            | Architect Classic                                                                                              |
| ----------------- | ---------------------------------------------------- | -------------------------------------------------------------------------------------------------------------- |
| Install           | Nothing — runs in the browser                        | Download and install (Windows, macOS, Linux)                                                                   |
| Protocol schema   | Schema 8 (latest)                                    | Schema 7 and earlier                                                                                           |
| Run interviews in | Fresco and Interviewer                               | Interviewer Classic (Interviewer and Fresco can automatically upgrade a protocol created in Architect Classic) |
| Backup & moving   | Download a `.netcanvas` file to keep or share a copy | Your `.netcanvas` file is already on disk — copy it anywhere                                                   |
| Updates           | Always the latest, automatically                     | Download each new release yourself                                                                             |
| Best for          | New studies (run in Fresco or Interviewer)           | Studies that must keep using the older Interviewer Classic app                                                 |
| Status            | Actively developed                                   | Maintenance mode — bug fixes only                                                                              |

<TipBox>

A **PWA** (Progressive Web App) is a website you can optionally install to your device. Architect runs in any modern web browser with nothing to install; installing it as a PWA adds quick access from your device and offline use.

</TipBox>

Architect can open a schema 7 protocol made in Architect Classic and upgrade it to schema 8, but this is one-way — a schema 8 protocol cannot be converted back or opened in Architect Classic. For a full explanation of schema versions and compatibility, see [Protocol Schema Information](/en/get-started/protocol-schema-information). If you are returning to Network Canvas after using the Classic apps, [Migrating from the Classic Apps](/en/get-started/migrating-from-classic) walks through the whole move — protocol, data, and day-to-day changes — end to end.
