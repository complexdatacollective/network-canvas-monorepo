Network Canvas is a suite of free, open-source tools for surveying networks,
designed around the needs of both researchers and their participants.
We provide a complete end-to-end workflow for networks research, with apps
for survey design and for interviewing. Using these tools, researchers can easily design, capture, and export network data.

## The research workflow

Every Network Canvas study has the same shape: you design a protocol that
describes your interview, you collect data by running that protocol with
participants, and you analyze the data you collected. The documentation is
organized around these three phases:

- **[Design Protocols](/en/design-protocols)** — decide what your interview
  will ask, and how, and express those decisions as a protocol file. Architect
  is the tool for building, previewing, and refining protocols.
- **[Collect Data](/en/collect-data)** — run your protocol as interviews with
  participants. Interviewer runs interviews on a device you control, while
  Fresco deploys your protocol to the web for participants to complete
  remotely.
- **[Analyze Data](/en/analyze-data)** — export the network data you collected
  in standard formats such as CSV and GraphML, ready for R, Python, or any
  other network-analysis package.

If you are still thinking your study through, start with
[Planning a Study](/en/get-started/planning-a-study/workflows), which walks
through the decisions to make before you begin.

## The applications

Each phase of the workflow is supported by one or more applications:

|                     | Runs where            | Access                                                          | Step in workflow |
| ------------------- | --------------------- | --------------------------------------------------------------- | ---------------- |
| Architect           | Browser, PWA          | [Open in browser](https://architect.networkcanvas.com)          | Build protocol   |
| Architect Classic   | Windows, macOS, Linux | [Download](/en/design-protocols/installing-architect-classic)   | Build protocol   |
| Interviewer         | Browser, PWA          | [Open in browser](https://interviewer.networkcanvas.com)        | Run interviews   |
| Interviewer Classic | Desktop & tablet      | [Download](/en/collect-data/interviewer/installing-interviewer) | Run interviews   |
| Fresco              | Browser (self-hosted) | [Deploy an instance](/en/collect-data/fresco/guide)             | Run interviews   |

_**PWA** (Progressive Web App): Architect and Interviewer run in any modern web
browser with nothing to install, and can optionally be installed to a device you
control for quick access and offline use._

<TipBox>

Not all apps are compatible. A protocol's schema version, determined by the tool it was
built in, controls which apps can open it, so consider compatibility when choosing
your tools. See
[Protocol Schema Information](/en/get-started/protocol-schema-information)
for a full explanation of schema versions, app compatibility, and automatic migration.

</TipBox>
