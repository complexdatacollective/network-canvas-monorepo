Interviewer is the desktop and tablet app for running Network Canvas interviews in person, guided by a trained interviewer on a device you control. It comes in two versions, and **both are supported** — you can install them side by side and use whichever suits your study. They run the same kinds of protocols and share the core interview experience; the main differences are the protocol schema each supports and the platform they're built on.

<KeyConcept title="Which version should I use?">

**Interviewer 8** (beta) is recommended for all new studies. It runs the latest **schema 8** protocols built in [Architect Web](/en/design-protocols/getting-started), and is built on the same modern platform as Fresco.

Existing **schema 7** protocols can be upgraded to **schema 8** automatically.

</KeyConcept>

|                 | Interviewer 6.x                      | Interviewer 8 (beta)                                                                                                                                   |
| --------------- | ------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Status          | Stable and supported; not going away | Beta — recommended for new studies                                                                                                                     |
| Protocol schema | Schema 7 and earlier                 | Schema 8 (latest)                                                                                                                                      |
| Built with      | The established desktop/tablet app   | The same new platform as Fresco 4.0                                                                                                                    |
| Connectivity    | Works fully offline                  | Offline-capable, but some schema-8 interfaces (e.g. [Geospatial](/en/design-protocols/interface-documentation/geospatial)) need an internet connection |
| New features    | Maintenance mode — bug fixes only    | Improved interview experience, new interfaces, and other enhancements                                                                                  |
| Best for        | Strictly offline settings            | New studies                                                                                                                                            |

The overall flow — installing protocols, starting interviews, and exporting data — is the same in each. For a step-by-step walkthrough of the participant and researcher experience, see [Using Interviewer](/en/collect-data/interviewer/using-interviewer). For guidance on choosing and setting up devices, see [Getting Started](/en/collect-data/interviewer/choosing-hardware).

For more on how schema versions affect which apps can open a protocol, see [App compatibility by schema version](/en/get-started/release-announcements) and [Protocol Schema Information](/en/get-started/advanced-topics/protocol-schema-information).
