<SummarySection>

The Protocol Gallery showcases publicly available research protocols
created using Network Canvas. These examples demonstrate how researchers
have implemented the platform across a range of study designs, populations,
and data collection contexts. By exploring the gallery, you can see how
different interfaces and features are combined in practice, review
associated metadata and documentation, and gain inspiration for your own
projects.

</SummaryCard>

## Search

Protocols are sorted by **most recently added** by default so that new examples are easier to discover.
Use the search bar to find protocols by study title, authors, or other searchable metadata. Search results update as you type.

## Filters

Filters can help you quickly identify protocols that use specific Network Canvas features or study designs.

- Uses Sociograms: Shows protocols that include a sociogram-based interface for collecting or visualizing relational data.
- Uses Rosters: Shows protocols that use predefined rosters as part of data collection.
- Uses Dyad Census: Shows protocols that use dyad census or tie-strength style data collection methods.

As the gallery grows, additional filters may be added to support more specific ways of browsing the protocols.

## Protocol detail pages

Selecting a protocol card opens a dedicated page for that protocol. These detail pages may include:

- Study title and authors
- Protocol summary
- Citation information
- Study metadata
- Population and field context
- Network data collection methods used
- Related files such as protocols or codebooks
- Links to associated resources when available

This structure makes it easier to compare protocols and understand how specific Network Canvas features are used in practice.

## Network Canvas in Practice

The features described in this tutorial are used across many different
research contexts, including public health studies, longitudinal cohort
studies, and implementation research projects.

To explore complete, real-world examples of Network Canvas deployments,
including study metadata and associated documentation, browse the
[Protocol Gallery](protocolgallery.networkcanvas.com).
