---
title: About Architect
---

Architect is the protocol designer for Network Canvas — the tool you use to build the interview protocols that run in Interviewer and Fresco. It comes in two versions that build the **same protocol**: **Architect Web**, which runs in your browser with nothing to install, and **Architect Desktop**, a downloadable application. The table below compares them — the main difference is how each one stores and moves your work.

<KeyConcept title="Not sure which to use?">

Choose **Architect Web** — it's the default for new studies and needs no setup.

[Open Architect Web →](https://architect.networkcanvas.com)

</KeyConcept>

|                      | Architect Web                                        | Architect Desktop                                            |
| -------------------- | ---------------------------------------------------- | ------------------------------------------------------------ |
| Install              | Nothing — runs in the browser                        | Download and install (Windows, macOS, Linux)                 |
| Where protocols live | In your browser, on this device                      | `.netcanvas` files on your computer                          |
| Backup & moving      | Download a `.netcanvas` file to keep or share a copy | Your `.netcanvas` file is already on disk — copy it anywhere |
| Updates              | Always the latest, automatically                     | Download each new release yourself                           |
| Best for             | New studies and most users                           | Existing file-based workflows; fully offline or archival use |
| Status               | Actively developed                                   | Maintenance mode — bug fixes only                            |
