# Network Canvas Homepage Media Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use
> superpowers:subagent-driven-development (recommended) or
> superpowers:executing-plans to implement this plan task-by-task. Steps use
> checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the marketing homepage's synthetic hero and product imagery
with authentic media, add the shared animated light background, and deliver one
accessible entrance sequence without changing page copy or structure.

**Architecture:** Add a small client-side `HeroIntro` composition that owns the
header/hero motion tree and decorative `BackgroundLights`. Keep media behavior
isolated in `HeroVideo`, keep product screenshot selection isolated in
`DeviceMockup`, and describe the animation contract with a pure helper that can
be tested without rendering Motion. Generated binary assets are acquired and
dimension-checked separately from the red-green component work.

**Tech Stack:** Next.js 16, React 19, TypeScript, Tailwind/Fresco theme tokens,
`motion/react`, `@codaco/art`, Vitest, Testing Library, Browser control, ffmpeg.

## Global Constraints

- Preserve the homepage's current section order, copy, links, and calls to
  action.
- Use 1600×1200 PNG screenshots from local Architect, local Interviewer, and
  the authenticated Fresco sandbox dashboard.
- Store screenshots in
  `apps/networkcanvas.com/public/images/screenshots/`.
- Store the original hero video at
  `apps/networkcanvas.com/public/videos/hero-video.mp4` and its poster at
  `apps/networkcanvas.com/public/images/hero-video-poster.jpg`.
- Use only existing theme CSS variables for `BackgroundLights` colors.
- Normal motion uses a short, controlled spring sequence; reduced motion uses
  no transforms, stagger, or autoplay.
- Do not introduce `any`, type assertions that bypass checking, ignore rules,
  barrel files, convenience re-exports, or unnecessary comments.
- Format every modified text/code file and resolve lint/type errors at their
  cause.
- `networkcanvas.com` is private and is listed in the changeset ignore list, so
  this change does not receive a changeset.

- [ ] **Step 4: Implement the pure entrance factory**

Create `lib/heroEntrance.ts`:

```ts
import type { Transition, Variants } from 'motion/react';

const entranceSpring: Transition = {
  type: 'spring',
  stiffness: 100,
  damping: 20,
  mass: 0.8,
};

export function createHeroEntrance(reduceMotion: boolean) {
  const pageVariants: Variants = {
    hidden: {},
    visible: {
      transition: { staggerChildren: reduceMotion ? 0 : 0.16 },
    },
  };
  const heroVariants: Variants = {
    hidden: {},
    visible: {
      transition: { staggerChildren: reduceMotion ? 0 : 0.12 },
    },
  };
  const itemVariants: Variants = reduceMotion
    ? {
        hidden: { opacity: 1, y: 0 },
        visible: {
          opacity: 1,
          y: 0,
          transition: { duration: 0 },
        },
      }
    : {
        hidden: { opacity: 0, y: 16 },
        visible: {
          opacity: 1,
          y: 0,
          transition: entranceSpring,
        },
      };

  return {
    initial: reduceMotion ? false : 'hidden',
    pageVariants,
    heroVariants,
    itemVariants,
  };
}
```

- [ ] **Step 5: Implement the top-of-page client composition**

Create `components/sections/HeroIntro.tsx`:

```tsx
'use client';

import { motion, useReducedMotion } from 'motion/react';
import { useEffect, useState } from 'react';

import { BackgroundLights } from '@codaco/art';
import { Header } from '~/components/layout/Header';
import { Hero } from '~/components/sections/Hero';
import { createHeroEntrance } from '~/lib/heroEntrance';

const lightColors = [
  'oklch(var(--neon-coral))',
  'oklch(var(--sea-green))',
  'oklch(var(--cerulean-blue))',
  'oklch(var(--mustard))',
];

export function HeroIntro() {
  const reduceMotion = useReducedMotion() ?? false;
  const entrance = createHeroEntrance(reduceMotion);
  const [mounted, setMounted] = useState(false);

  useEffect(() => setMounted(true), []);

  return (
    <div className="relative isolate overflow-hidden">
      {mounted ? (
        <div className="pointer-events-none absolute inset-0 z-0 opacity-20">
          <BackgroundLights
            large={1}
            medium={3}
            small={0}
            speedFactor={0.8}
            colors={lightColors}
          />
        </div>
      ) : null}
      <motion.div
        className="relative z-10"
        variants={entrance.pageVariants}
        initial={entrance.initial}
        animate="visible"
      >
        <Header entranceVariants={entrance.itemVariants} />
        <Hero
          containerVariants={entrance.heroVariants}
          itemVariants={entrance.itemVariants}
        />
      </motion.div>
    </div>
  );
}
```
